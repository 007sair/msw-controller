📋 请求记录

页面发起的 HTTP 请求会自动记录，包括请求方法、URL 和时间戳。

• 点击请求记录可切换对应 Handler 的启用状态

• 绿色边框表示已匹配，灰色边框表示未匹配

⚙️ Handlers

MSW 中定义的请求处理器，用于拦截和模拟 API 响应。

• 启用的 Handler 会拦截匹配的请求并返回模拟数据

• 禁用的 Handler 不会拦截请求，让请求发送到真实服务器

💡 使用技巧

• 使用搜索功能快速找到特定的 Handler 或请求

• 通过启用/禁用 Handler 来测试不同的数据场景

• 利用实时更新功能监控页面的网络活动
